## API Request/Response Formats

### Calendar Events

```typescript
// POST /api/calendar/events
Request: {
  title: string,
  date: string,           // ISO date
  time?: string,          // HH:MM format
  type: "deadline" | "meeting" | "appointment" | "reminder",
  emailId?: string,
  description?: string,
  reminderMinutes?: number
}

Response: {
  event: {
    id: string,
    userId: string,
    ...requestData,
    createdAt: string
  }
}
```

### Email Search

```typescript
// POST /api/search/emails
Request: {
  query?: string,
  sender?: string,
  subject?: string,
  dateFrom?: string,
  dateTo?: string,
  hasAttachment?: boolean,
  isStarred?: boolean,
  category?: string
}

Response: {
  emails: Array<{
    id: string,
    subject: string,
    from: string,
    date: string,
    snippet: string
  }>,
  query: string
}
```

### Team Assignments

```typescript
// POST /api/team/assignments
Request: {
  emailId: string,
  assignedTo: string,
  deadline?: string,
  notes?: string,
  priority?: number
}

Response: {
  assignment: {
    emailId: string,
    assignedTo: string,
    assignedBy: string,
    deadline?: string,
    status: "assigned",
    notes: string[],
    priority: number,
    createdAt: string
  }
}
```
